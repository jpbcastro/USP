compilador: gcc (Ubuntu 7.5.0-3ubuntu1~18.04) 7.5.0

A entrada deve ser uma frase sem espaços como "olatudobem?" ou "ola_tudo_bem?".
Os caracteres corrompidos durante o meio de transmissão e identificados são subtituidos por #.
A variável global do tipo inteira "TipoDeControleDeErro" determina qual controle de erro será usado, sendo 0 para bit de paridade par, 1 para bit de paridade ímpar, 2 para CRC.
